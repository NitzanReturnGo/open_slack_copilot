# Example Skills

Copy any of these into `~/.open_slack_copilot/skills/reply/<skill_name>/SKILL.md` to use them.

| Skill | Description |
|---|---|
| [polite_reply](polite_reply.md) | Warm, professional tone with empathy for frustration |
| [technical_review](technical_review.md) | Precise, constructive feedback on code and architecture |
| [concise_summary](concise_summary.md) | Short 2-3 sentence replies, straight to the point |
| [customer_support](customer_support.md) | Empathetic support replies with clear next steps |

### Default skill override

To override the built-in default instruction, copy [default.md](default.md) to `~/.open_slack_copilot/skills/reply/default.md` and edit to taste.
