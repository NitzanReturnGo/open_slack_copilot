# M15 — Follow-ups use case

[← Back to PRD](../../PRD.md)

## Story

A manager posts in a channel (often mentioning a **user group**) asking people to complete something. In the **thread**, they invoke CoPilot (e.g. `@CoPilot please follow up`). CoPilot should infer how often to re-check, who to track, what counts as “done,” register a **recurring** check via [`schedule_prompt`](../../../common/tools/schedule_tool.py#L13-L45) ([`cron`](../../../common/tools/schedule_tool.py#L33-L36), [`expires_in_days`](../../../common/tools/schedule_tool.py#L37-L40)), and on each run **[`send_dm_as_app`](../../../common/tools/send_dm_as_app.py)** to anyone who has not completed, with a link back to the thread.
